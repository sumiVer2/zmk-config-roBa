# zmk-config-roBa

## Layer Structure
```
0: DEFAULT
   通常入力レイヤー。
   英字入力を中心に、親指操作で各レイヤーへ即座に遷移可能。

1: SYMBOL
   記号入力専用レイヤー。
   プログラミングや文章入力で頻出する記号を、
   Shift を使わずに入力できる構成。

2: NUMBER
   数字・Fn レイヤー。
   右手側をテンキー、左手側を F1〜F12 とし、
   数値入力と操作キーを高速に扱える。

3: MOVE
   カーソル移動・操作レイヤー。
   矢印キーや Home / End を集約し、
   マウスに頼らない編集操作を可能にする。

4: BLUETOOTH
   Bluetooth 接続管理レイヤー。
   接続先切り替えやペアリング解除など、
   通常操作と明確に分離した管理用レイヤー。

5: SCROLL
   スクロール専用レイヤー。
   左キーをホールドすることで、
   トラックボール操作をスクロール入力として使用可能。
```

## Other Settings
- ロータリーエンコーダーは逆スクロールに設定
- オートマウスレイヤー切り替え機能無効化
- Lホールドでトラックボールマウスでのスクロールオン
